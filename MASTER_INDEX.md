## ğŸ“ Usage Instructions

### For Website Implementation
1. Copy JSON-LD schemas from `seo/` and `geo/` folders
2. Add to `<head>` section of product pages
3. Submit sitemap.xml to Google Search Console
4. Validate with Google's Rich Results Test

### For Ecommerce Platforms

**Shopify:**
```
1. Go to Products > Import
2. Upload shopify_import.csv
3. Map image URLs
4. Review and publish
```

**WooCommerce:**
```
1. Install WP All Import plugin
2. Upload woocommerce_import.csv
3. Map fields
4. Import images
```

**Amazon:**
```
1. Login to Seller Central
2. Go to Inventory > Add Products via Upload
3. Upload amazon_import.csv
4. Follow template guidelines
```
